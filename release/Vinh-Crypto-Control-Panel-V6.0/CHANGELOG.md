# Changelog

## V6.0

- Renamed product to Vinh Crypto Control Panel.
- Added PWA manifest, service worker, and SVG icon.
- Added offline shell and online/offline indicator.
- Added docs, handover notes, test cases, and release notes.
- Added release package folder.

## V5.1

- Added snapshot history.
- Added decision journal.
- Added localStorage export/import JSON.
- Added import validation and confirmation before overwrite.

## V5.0

- Added BTC Treasury mode.
- Added portfolio allocation, total USD, total BTC equivalent, and risk score.
- Added DCA BTC panel with non-all-in reminder.

## V4.2

- Added default personal watchlist: BTC, OKB, ATOM, ETH, SOL, USDT.
- Added add, remove, and reset watchlist controls.
- Added watchlist table with data status.

## V4.1

- Fixed SHIB CoinGecko mapping.
- Added OKB and USDT mappings.
- Replaced all-or-nothing API loading with independent settled loading.
- Added market cache with 180 second TTL and LIVE/CACHE/STALE status.
- Removed 30 second auto refresh.
- Reworded trading language into safer accumulation and risk-management language.
