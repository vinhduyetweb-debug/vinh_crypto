# Release Notes V6.0

Vinh Crypto Control Panel V6.0 upgrades the previous crypto signal dashboard into a static personal control panel.

## Included

- Safer wording: accumulation, monitoring, and risk reduction instead of aggressive trading language.
- Independent API failure handling.
- localStorage market cache with LIVE/CACHE/STALE labels.
- Personal watchlist.
- BTC Treasury and DCA planning.
- Snapshot history and decision journal.
- JSON backup and restore.
- PWA manifest, service worker, offline shell, and online/offline indicator.

## Not Included

- No backend.
- No login.
- No wallet connection.
- No private API key.
- No auto trade.
- No futures/leverage workflow.
